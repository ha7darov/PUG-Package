## How to use Pug?

### Installing pug using package

- ### Create the first `index.pug` file


- ### and type this command in your terminal and index.pug file will be compiled

### Comand Compiler
```bash
npm run pug
```
### or
```bash
pug -w ./ -o ./src -P
```

#### The `src` in this code works as a word file
